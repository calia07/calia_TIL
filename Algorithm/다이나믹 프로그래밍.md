# 다이나믹 프로그래밍 (Dynamic Programming) = 동적 계획법
#### 메모리를 적절히 사용하여 수행 시간 효율성을 향상시키는 방법
#### 이미 계산한 결과는 별도의 메모리에 저장하여 다시 계산하지 않도록 함

<br>

## 1. 필수 개념
- ### 다이나믹 프로그래밍의 조건
  - 최적 부분 구조 : 큰 문제를 작은 문제로 나누고 이를 통해 큰 문제의 해결이 가능하다.
  
  - 중복 되는 부분 문제 : 동일한 작은 문제를 반복적으로 해결한다. 
  - ex) 피보나치 수열
    - 점화식 : 인접한 항들 사이의 관계식
